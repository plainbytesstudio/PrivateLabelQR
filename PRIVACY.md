# Privacy Policy for Private Label QR

**Last Updated: March 2025**

PlainBytes Studio ("we," "us," or "our") respects your privacy. This Privacy Policy explains our practices regarding data collection and usage for the **Private Label QR** Windows application.

### 1. 100% Offline Processing
Private Label QR is designed as a **local-first** utility. All core functions, including QR code generation, Barcode generation, and PDF label rendering, are performed entirely on your local machine.

### 2. No Data Collection
*   **No Personal Data:** The app does not collect, store, or transmit any personal information (such as names, emails, or IP addresses).
*   **No Usage Tracking:** We do not use any analytics, telemetry, or tracking scripts to monitor how you use the application.
*   **No Content Logging:** The data you input (URLs, text, or asset numbers) and the files you import (Excel/CSV) are processed in memory and never sent to any server or external service.

### 3. File System Access
The application requires your permission to access the file system only for the following manual actions:
*   **Importing:** To read Excel or CSV files selected by you for bulk generation.
*   **Exporting:** To save generated QR images or PDF label sheets to a folder of your choice.

### 4. Third-Party Services
The application does not integrate with any third-party APIs, advertising networks, or cloud services. There is no background network communication.

### 5. Security
Because the application runs 100% offline, your data is protected by the security of your own device and is never exposed to the internet through this software.

### 6. Compliance
This policy is designed to comply with global privacy standards, including **GDPR** and **CCPA**, by virtue of not collecting data.
